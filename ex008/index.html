Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> e </b> (Não Semântica).

Nesta frase temos um termo em destaque usando a tag <strong> </strong> (Semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a TAG <i></i> (Não Semântica).

Nesta frase temos um termo em ênfase usando a TAG <em></em> (Semântica).

Texto Marcado

Podemos criar um texto marcado usando a TAG <mark></mark>.

Para alterar a cor da marcação, pode ser usado um style dentro do marcador "MARK" ou abrir uma tag em CSS e fazer a modificação do que estiver na marcação "MARK".

Texto grande e Pequeno

Estamos criando um texto com palavras grandes e também com palavras pequenas neste parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas, não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Também pode ser utilizado para sublinhar, a TAG <u></u>

Texto Sobrescrito

Para inserir coisas do tipo x20+3
